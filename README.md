# JobTracker

## Descripción

JobTracker es una aplicación web diseñada para ayudar a gestionar procesos de búsqueda de empleo, permitiendo registrar vacantes, realizar seguimientos, administrar usuarios y visualizar estadísticas mediante un dashboard.

Este proyecto está siendo desarrollado como práctica de desarrollo web, aplicando conceptos de HTML, CSS, JavaScript y, en futuras versiones, backend con Node.js y bases de datos relacionales.

---

## Funcionalidades actuales

* Dashboard con estadísticas generales.
* Gestión de vacantes.
* Gestión de usuarios.
* Gestión de seguimientos.
* Perfil de usuario.
* Interfaz responsive.
* JavaScript Vanilla.
* Frontend contenido en un único archivo HTML.
